This plugin fixes the WordPress admin menu which is caused by the latest version of Google Chrome (45).

== Description ==
Has your WordPress admin menu gone all weird? This isn't to do with your website but your Google Chrome browser. Use this plugin to fix the issue by simply activating it.

== Installation ==
1. Upload the `chrome-admin-menu-fix` plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!

== Frequently Asked Questions ==
= How does this work? =
The plugin simply adds a snippet of CSS to your WordPress admin menu style to avoid the browser conflict.
